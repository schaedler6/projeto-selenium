# Projeto Selenium - Automated Tests

Test automation project using Selenium WebDriver and Python.

## Overview

This repository contains browser automation tests focused on login validation and web interaction practice.

Main goals:
- practice automated testing
- improve Selenium skills
- organize scripts and test structure
- simulate real QA and automation scenarios

## Technologies

- Python 3
- Selenium WebDriver

## Test Scenarios

- Successful login validation
- Failed login validation
- Navigation and interaction practice
- Basic report-oriented structure

## Use Cases

This project is useful for:
- learning test automation
- practicing browser automation
- building QA portfolio projects
- understanding automated validation flows

## Requirements

- Python 3.x
- Selenium
- Compatible browser driver
